NS POS - Demo React + Vite project
===================================

This is a prototype point-of-sale (POS) web app for beverage sales (NS POS).
It uses Tailwind for styling (configured), Recharts for the bar chart, and lucide-react for icons.

How to run (locally):
1. unzip and cd into project folder
2. npm install
3. npm run dev
4. Open the local dev url shown (http://localhost:5173)

Demo login phones:
- 0912345678 (Vicky)
- 0900000000 (Admin)

Notes:
- Data is stored in localStorage (no backend).
- Export CSV available on Report page.
